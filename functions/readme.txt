
The "functions" folder is the build target for the AWS functions.
The source code is in src/api.
Each function is a separate file, which must export a 'handler' function.

See https://www.netlify.com/docs/functions/
and https://github.com/netlify/netlify-lambda
and https://docs.aws.amazon.com/lambda/latest/dg/nodejs-prog-model-handler.html
